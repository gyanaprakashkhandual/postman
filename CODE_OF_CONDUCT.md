# Code of Conduct

---

## Our Commitment

This project is open to everyone. We are committed to maintaining a welcoming, respectful, and harassment-free environment for all contributors, regardless of age, background, level of experience, gender, identity, nationality, appearance, race, religion, or any other personal characteristic.

We ask everyone who participates — through issues, pull requests, discussions, or any other form of engagement — to uphold this standard.

---

## Expected Behavior

When participating in this project, please:

- Be respectful and considerate in all communication
- Welcome contributors of all experience levels
- Accept constructive feedback graciously
- Focus on what is best for the project and the community
- Give credit where it is due
- Keep discussions relevant to the project

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, intimidation, or discrimination of any kind
- Personal attacks or insults directed at contributors
- Publishing private or personal information about others without consent
- Trolling, inflammatory remarks, or sustained disruption of discussions
- Any conduct that would be considered inappropriate in a professional setting
- Attempting to exploit, probe, or abuse the live API infrastructure associated with this project

---

## Scope

This Code of Conduct applies to all project spaces, including the GitHub repository, issues, pull requests, and any other communication channels related to this project.

---

## Reporting a Violation

If you witness or experience behavior that violates this Code of Conduct, please report it by opening a private issue or contacting the repository owner directly via GitHub.

Reports will be reviewed promptly, treated with confidentiality, and addressed appropriately. The safety and well-being of community members is the priority.

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. In response to a violation, maintainers may take actions including:

- A private warning to the individual
- Removal of the offending comment, issue, or pull request
- Temporary or permanent ban from contributing to the project
